# Set Up 

## Data 
We use the data from Lennarts [NatComm Paper](https://doi.org/10.1038/s41467-021-21970-2)

## Decoding Results 
You can find the data and masks on hummel in
`/beegfs/g/schuck/bbf2463/Projects/Retreat2026`

You'll find the decoding results further in 
`.../Data/highspeed-decoding/decoding`
for each subject. 
Each sub-folder includes the subfolders `data` which include the csv-files `sub-XX_decoding.csv` and `sub-XX_thresholding.csv`

## Virtual Environment
On Hummel you can access the virtual environment via: 
```
source /beegfs/g/schuck/bbf2463/Projects/Retreat2026/miniforge3/bin/activate
conda activate replay
```
